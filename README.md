# hydro-tap

## Synopsis

Tap formatter for [hydro](https://github.com/hydrojs/hydro)

## Installation

npm:

```bash
npm install hydro-tap
```

component:

```bash
component install hydrojs/hydro-tap
```

Standalone:

```html
<script src="hydrojs-tap.js></script>
```
